# glhera-query

**glhera-query** é uma biblioteca frontend para sincronizar data de fetch como do [TanStack Query](https://tanstack.com/query/latest/docs/framework/react/overview) mais conhecido como **React Query**

A diferença entre o **glhera-query** e o **TanStack Query** é que o **glhera-query** é **signal based** e não faz cache por padrão (somente em queries infinitas)

Por conta de ser **signal based** o **glhera-query** pode ser extremamente simples e mais fácil de usar do que o **TanStack Query**
